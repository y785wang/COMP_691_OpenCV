
---------- Program running environment ----------

The program is written by using CLion on Mac.
The program can be imported directly as new cMake project by using CLion.

---------- Program major structure  ----------

There are 4 .cpp files and 3 .h files.
	1) main (takes no arguments)
	2) featureDetection
	3) featureDescription
	4) FeatureMatch

The testing image addresses are hard coded in main.cpp file on line 19 (image one) and lines 48~52 (image two).

For each class there is a corresponding brief comment to explain who it works and what it does. Some functions are quite complicated, for the detail, see report.s
